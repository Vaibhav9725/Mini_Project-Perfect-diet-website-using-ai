# The Perfect Diet Website

An AI-powered diet and nutrition website that helps users find their perfect diet plan based on their health goals and lifestyle.

---

## Features

- AI-powered personalized diet recommendations
- Healthy meal plans and nutrition tips
- Calorie and nutrient tracking
- Diet plans based on fitness goals (weight loss, muscle gain, etc.)
- Fully responsive design for all devices
- Smooth animations and modern UI

---

##  Technologies Used

- **HTML5** — Structure of the website
- **CSS3** — Styling and layout
- **JavaScript** — Interactivity and logic
- **Bootstrap** — Responsive design framework
- **jQuery** — DOM manipulation
- **Font Awesome** — Icons
- **AI Integration** — Personalized diet suggestions
